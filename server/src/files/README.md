# Files Module

Модуль для работы с файлами в NestJS приложениях. Обеспечивает сохранение и удаление файлов с автоматическим созданием директорий и подробным логированием.



## Описание

Модуль предоставляет сервис FilesService для управления файлами. Поддерживает сохранение файлов с генерацией уникальных имен, создание статической директории, удаление файлов и полное логирование всех операций.

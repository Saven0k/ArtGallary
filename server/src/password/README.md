# Password Module

Модуль для безопасной работы с паролями в NestJS приложениях с использованием bcrypt и Winston логированием.


## Описание
Модуль предоставляет сервис PasswordService с тремя основными методами для работы с паролями. Все операции логируются через Winston с детальным контекстом.

### Методы

#### hashPassword(password: string): Promise<string>

- Хэширует пароль с использованием bcrypt

- Использует 10 раундов соли

- Возвращает хэшированную строку

#### comparePassword(password: string, hashPassword: string): Promise<boolean>

- Сравнивает пароль с хэшем

- Возвращает true при совпадении

- Обрабатывает ошибки с логированием

#### validatePasswordStrength(password: string): { isValid: boolean; message?: string }

- Проверяет пароль на сложность

- Критерии: минимум 6 символов, максимум 50, цифры, оба регистра, спецсимволы (!@#$%^&*)

- Возвращает объект с результатом и сообщением об ошибке

## Требования к паролю
* Минимальная длина: 6 символов

* Максимальная длина: 50 символов

* Как минимум одна цифра

* Буквы в верхнем и нижнем регистре

* Как минимум один спецсимвол: !@#$%^&*

## Безопасность

* Пароли никогда не логируются в открытом виде

* Используется bcrypt с солью

* Строгая валидация перед хэшированием

* Обработка ошибок с детальным логированием
